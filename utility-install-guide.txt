ProServ Utilities 1.0 - Installation Guide
-----------------------------------
Prerequisite:

a. Java version >= 1.8
b. Text Editor

Input(s): 
Please provide the below inputs to have it bundled within the artifacts:

a. IntegratorKey
b. Admin User Guid (either with Admin or User Consent)
c. Secret Key
d. Private/Public RSA Key pair
e. Input CSV with envelopeId,reminderEnabled,reminderDelay,reminderFrequency,expireEnabled,expireAfter,expireWarn columns
f. Proxy Port or Host if needed to call external APIs from your system
g. Account Guid
h. profileDesc.properties file bundled with other artifacts
i. Provide inDirpath where input CSV is present and outDirpath where system will 
   write the output CSVs

Artifacts:
1) proServUtilities-1.0.jar
2) ps_oauth_process_demo.bat (For Demo)
3) ps_oauth_process_prod.bat (For Prod)

Installation Steps:
1) Copy all the artifacts to a location on your server - [dir]
2) Open ps_oauth_process_*.bat (* represents demo or prod) in a text editor and
   update the above required inputs
3) Execute ps_oauth_process_*.bat (* represents demo or prod) in [dir] using command window.

Output:
a. outDirpath will have output CSVs with results for each envelopeIds success or failure
b. proServUtilities.log will be created for each run
